# İmza Gayrimenkul - Web Application

Lüks gayrimenkul portföy yönetimi ve müşteri ilişkileri portalı.

## Proje Yapısı (GÜNCELLENMİŞ)

Proje aşağıdaki ana bileşenlerden oluşmaktadır:

- **backend/**: Flask çekirdek ve modüler Blueprint yapısı (Core & Addons).
- **frontend/**: Kamu sitesi (investment), mahalle portalı (neighborhood) ve admin paneli (portal).
- **infrastructure/**: Config, docs, migrations ve yardımcı scriptler.
- **static/**: Ortak kullanılan statik dosyalar.
- **shared/**: Veri modelleri ve servisler.
